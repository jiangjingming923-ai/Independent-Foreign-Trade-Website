# Independent Foreign Trade Website

A multilingual B2B foreign trade website built with Next.js and Tailwind CSS. It includes a product catalog, product detail pages, company pages, inquiry forms, SEO metadata, sitemap/robots generation and structured data.

## Features

- Static export ready for Vercel, Netlify or static hosting.
- English and Chinese routes under `/en` and `/zh`.
- Pages for Home, Products, Product Details, About, Contact and Inquiry.
- SEO metadata, sitemap, robots and JSON-LD structured data.
- Lightweight product configuration in `data/products.ts` for easy catalog updates.
- Netlify-compatible inquiry form markup.

## Getting Started

```bash
npm install
npm run dev
```

Build the static site:

```bash
npm run build
```

The static export is generated by Next.js because `next.config.ts` sets `output: 'export'`.

## Product Management

Edit `data/products.ts` to add, remove or update products. Each product contains multilingual names, descriptions, features, specs, market coverage and an image path.

## Deployment Notes

- **Vercel:** import this repository and use the default Next.js build command.
- **Netlify:** use `npm run build`; the static output directory is `out`.
- **WordPress:** deploy the static export separately and link/embed pages, or migrate the page sections into a WordPress theme while keeping `data/products.ts` as the catalog source during development.
